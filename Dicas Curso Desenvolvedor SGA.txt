Aula 47, 48, 49, 50, 51, 52

47
IN -> Permite especificar vários valores em uma cláusula WHERE.É a abreveatura do OR.
SELECT column_name(s) FROM usuarios WHERE column_name IN (value1...);

48
BETWEEN -> É usado para selecionar valores dentro de um intervalo.
SELECT column_name(s) FROM table_name Where column_name BETWEEN value1 AND value2;

49
Aliaes -> Fornece um nome temporário, que só existe durante a duração da consulta.
SELECT column_name As aliases_name FROM usuarios;

50
INNER JOIN -> Seleciona registros que possuem valores corespondentes em ambas as Tabelas.
SELECT column_name(s)FROM table1 INNER JOIN table 2 ON table1.column_name = table2.column_name.

51
LEFT JOIN -> Obtem os dados relacionados e não relacionados de duas tabelas.
SELECT column_name(s) FROM table2 ON table1.column_name = table2.column_name;

52
RIGHT JOIN -> Obtem os dados relacionados e não relacionados de duas tabelas.
SELECT column_name(s) FROM table2 ON table1.column_name = table2.column_name;
